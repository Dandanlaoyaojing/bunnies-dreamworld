## 📝 技术细节

### 跳转方法
```javascript
// 账户管理页面
myNotes() {
  wx.navigateTo({
    url: '/pages/my-notes/my-notes'
  })
}

// 我的页面
goToMyNotes() {
  if (!this.data.userInfo.isLoggedIn) {
    this.goToLogin()
    return
  }
  
  wx.navigateTo({
    url: '/pages/my-notes/my-notes'
  })
}
```
