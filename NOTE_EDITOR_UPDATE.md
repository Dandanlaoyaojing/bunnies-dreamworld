## 🚀 使用流程

1. **进入编辑**: 点击首页"记笔记"图标
2. **输入标题**: 在标题输入框中输入或自动生成
3. **选择分类**: 点击分类图标选择笔记分类
4. **编辑内容**: 
   - 直接输入文字
   - 添加图片并OCR识别
   - 录制语音并转文字
5. **管理标签**: AI生成或手动添加标签
6. **配置保存**: 选择是否保存附件
7. **保存笔记**: 点击保存按钮完成
